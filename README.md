### Bottle App Template
- `run.py` requires:
  - `-a` : host address
  - `-p` : port
  - `-s` (optional) : smtp sender
  - `-m` (optional) : smtp receiver
- use `launcher.sh` to run with presets
- to install non-dev dependencies : `npm install --production`
- for dev use:
  - run `npm install`
  - scss -> css transpiler : `npm run scss`
  - tsc -> js transpiler : `npm run tsc`
